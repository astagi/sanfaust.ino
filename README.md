# Sanfaust.ino

Cool addon for your belt, only for love hunters!

## Hardware Requirements

- Arduino Board (I use Arduino Uno)
- HD44780 LCD

## How to configure

	$ ./configure.sh

You can use sanfaust/cfg.h generated file to configure your project.
